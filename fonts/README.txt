Place a math-capable TrueType/OpenType font in this folder named `math_font.ttf`.

Recommended free fonts:
- STIX Two Math (STIXTwoMath.otf) — https://www.stixfonts.org/
- Latin Modern Math (lm-math.otf) — available via TeX distributions

Rename your chosen font to `math_font.ttf` for the generator to pick it up automatically.
Ensure the font's license permits redistribution if you plan to include it in the repository.
